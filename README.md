# 弑君者 在线原型

这是根据 `桌游设计改ue5-弑君者.xlsx` 里的规则制作的浏览器版可玩原型。

## 已实现

- 4 人局和 5 人局开局。
- J、Q、K 共 12 个 Boss，按 J -> Q -> K 出现，同点数内花色随机。
- 玩家手牌上限：4 人局 5 张，5 人局 4 张。
- 出牌规则：单张、对子点数和不超过 10、A 加任意一张。
- 黑桃削弱 Boss 攻击、红桃回收弃牌、梅花伤害翻倍、方块轮流抽牌。
- Boss 花色压制对应玩家花色效果。
- 防御阶段弃牌点数必须大于等于 Boss 攻击力。
- 正常击杀和完美击杀。
- 5 人局技能卡：无效 Boss 花色压制，并指定下一名玩家。

## 运行

```bash
npm start
```

也可以直接用任意静态服务器打开项目根目录。
